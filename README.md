🖼️ Image Steganography Web Tool

A simple browser-based tool that hides files inside an image using basic steganography.

This project allows users to encode (hide) a file inside an image and later decode (extract) it using a passcode.
Everything runs directly in the browser using HTML, CSS, and JavaScript — no server required.

⚙️ How It Works

Encode (Hide File)

1. Select a cover image
2. Select the file you want to hide
3. Enter a passcode
4. Click Encode & Download
5. The system creates a new image containing the hidden file

Decode (Extract File)

1. Upload the encoded image
2. Enter the same passcode
3. Click Decode & Extract
4. The hidden file downloads as a ZIP file
